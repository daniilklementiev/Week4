Сериализация (Serialize)
от англ. serial - последовательный
процесс представления объекта в виде последовательности (обычно строки), по которой этот объект может быть однозначно восстановлен

Наиболее распространенные задачи сериализации:
сохранение
передача по сети

В C# наиболее популярным следующие технологии:
Бинарная сериализация
устаревшая/не рекомендуется
требует протокола Stream/FileStream
требует атрибута [Serializable] для класса
информация о типах сохраняется в файле с данными

XML сериализация
может работать с StreamReader/StreamWriter
сериализует только объекты public классов
при создании сериализатор привязывается к типу данных, в файл информация о типах не добавляется
не требует атрибута [Serializable] для класса
! не сериализует коллекции IDictionary

JSON сериализация
реализована статическими методами
работает со строками, их сохранять/читать нужно отдельно
сериализует только public свойства. В .NET 5+ можно включить сериализацию полей атрибутом [JsonInclude]
private свойства не сериализуются
сериализует коллекции IDictionary
